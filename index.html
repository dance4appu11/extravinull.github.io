How does it work?

What is ReShade? It is a generic post-processing injector for games and video software developed by crosire, for more info please visit https://reshade.me/

Available On Windows 7+

Great Community.

Open Source.

How to install the presets :)

1. Download Extravi's ReShade Preset.
2. Open the setup file.
3. Click browse.
4. Run the installer.
5. Perfect! You're set!
6. To toggle the shaders, the hotkeys are F8 OR FN + F8 and F7
